## Usage

1. **Clone the repository:**
   ```bash
   git clone https://github.com/MtJakhar/coffeeShop.git
   cd coffeeShop
   ```

2. **Install Dependencies:**
   ```bash
   npm install
   ```

3. **Run the Application:**
   ```bash
   npm run dev
   ```

4. **Open in Browser:**
   Visit [http://localhost:3000](http://localhost:3000) in your browser.
